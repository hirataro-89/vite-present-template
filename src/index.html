Vite Static Site Template

そのまま使える、
Webサイト制作テンプレート。

Vite + Sass + モダンCSSのミニマルな制作環境です。
設定ファイルと本文を書き換えるだけで、あなたのサイトとして公開できます。

使い方

以下の4ステップで、このテンプレートをあなたのサイトとして公開できます。
詳しい手順は、同梱の README.md をご覧ください。

  1. 1

    site.config.json を編集してサイト名・タイトル・OGPなどを設定

  2. 2

    src/index.html の本文を自分のコンテンツに書き換える

  3. 3

    src/scss/base/_variables.scss でブランドカラーを変更

  4. 4

    yarn build でビルドして dist/ をサーバーにアップロード
